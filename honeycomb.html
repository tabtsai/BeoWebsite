    Adobe Illustrator

    Learning the very vector fundamentals

    Since its release, Adobe's Illustrator has grown to become the designer's right hand man when it comes to creating premier vector graphics and elements. There's so much you can do with the tool that it can be difficult to know where to start.

    On this course we will be introducing you to the program and covering some essential practice exercises.

    Along the way we will be covering many topics, such as how to set up your documment, understand how the program works, use text in Illustrator, prepare and import images into Illustrator and finally export your document ready for print.

    Topics Covered

    • What is Illustrator
    • Document Setup
    • Using text
    • Importing images
    • Creating shapes
    • Export the document
